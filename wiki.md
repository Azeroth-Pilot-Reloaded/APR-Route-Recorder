## Action / Progression Options

| Option            | Description                                                                                                                                                                                                                                                                                                                                                                         | Expected Syntax                                                                                             |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `ExitTutorial`    | Exile’s Reach specific step that auto-skips if the exit quest is no longer tracked.                                                                                                                                                                                                                                                                                                 | `ExitTutorial = 59985`                                                                                      |
| `PickUp`          | List of quest IDs to pick up. The step remains active until all quests are in the log. **If using `PickUpDB`, you must also include the base `PickUp` field with at least one of the related quest IDs.**                                                                                                                                                                           | `PickUp = { 39688 }`                                                                                        |
| `PickUpDB`        | Alternative quest IDs (class/faction variants). The step validates if any listed quest is active or completed. Requires the main `PickUp` field for compatibility.                                                                                                                                                                                                                  | `PickUpDB = { 39688, 39694, 40255, 40256 }`                                                                 |
| `DropQuest`       | Quest obtained from a mob drop. **Must always be paired with a corresponding `DroppableQuest` definition.**                                                                                                                                                                                                                                                                         | `DropQuest = 48876`                                                                                         |
| `DroppableQuest`  | Defines the mob and quest drop behavior. Can be added to any step (same as `Fillers`). When not paired with `DropQuest`, it acts as a passive filler — the addon will display the target mob but will not block progression if the quest isn’t dropped yet.                                                                                                                         | `DroppableQuest = { Qid = 41234, MobId = 133713, Text = "Fel Marauder" }`                                   |
| `Qpart`           | Quest objectives to complete, mapped by quest ID and objective index.                                                                                                                                                                                                                                                                                                               | `Qpart = { [12345] = { 1, 2 } }`                                                                            |
| `QpartDB`         | Alternative quest IDs for the same `Qpart` block. The step must also contain a base `Qpart`.                                                                                                                                                                                                                                                                                        | `QpartDB = { 12345, 12346 }`                                                                                |
| `QpartPart`       | Variant of Qpart that splits a single objective into several sub-parts to guide the player precisely. TrigText is mandatory and should indicate the current sub-objective. It can now handle both fraction format ("1/3") and percentage format ("33%", "66%", "100%"). Commonly used to direct the player to multiple interaction points or elements for the same quest objective. | `QpartPart = { [12345] = { 1 } }, TrigText = "1/3"`                                                         |
| `Fillers`         | Optional side objectives (same syntax as `Qpart`). Fillers are quests you can progress during any step but are not required to complete it. When a step later defines a matching `Qpart`, any already completed filler objectives are automatically validated.                                                                                                                      | `Fillers = { [49529] = { 1 }, [49897] = { 1 } }`                                                            |
| `Done`            | Quests to turn in. The step completes once all listed quests are handed in. **If using `DoneDB`, the base `Done` field must also be included.**                                                                                                                                                                                                                                     | `Done = { 12345, 12400 }`                                                                                   |
| `DoneDB`          | Alternative quest IDs counted as the same completion. Requires `Done`.                                                                                                                                                                                                                                                                                                              | `DoneDB = { 12345, 54321 }`                                                                                 |
| `Treasure`        | Treasure or vignette quest. The step skips once the ID is flagged as completed.                                                                                                                                                                                                                                                                                                     | `Treasure = 56789`                                                                                          |
| `Group`           | Marks an optional group quest and shows a popup (`QuestId`, `Number`).                                                                                                                                                                                                                                                                                                              | `Group = { questID = 51384, Number = 3 }`                                                                   |
| `GroupTask`       | Associates the step with `WantedQuestList` to store the player’s decision for a group quest.                                                                                                                                                                                                                                                                                        | `GroupTask = 51384`                                                                                         |
| `Achievement`     | Fine tracking of an achievement criterion. The step validates once the specified criteria is completed.                                                                                                                                                                                                                                                                             | `Achievement = { achievementID = 61576, criteriaIndex = 1 }`                                                |
| `Scenario`        | Fine tracking of a scenario criterion (`scenarioID`, `stepID`, etc.). Can also be tied to a quest ID for progression consistency.                                                                                                                                                                                                                                                   | `Scenario = { criteriaID = 106007, criteriaIndex = 1, scenarioID = 3101, stepID = 15911, questID = 86820 }` |
| `EnterScenario`   | Guides the player to the scenario entrance (map ID). Usually tied to a quest.                                                                                                                                                                                                                                                                                                       | `EnterScenario = { questID = 86636, mapID = 2502 }`                                                         |
| `DoScenario`      | Indicates that the player should complete the scenario. Usually tied to a quest + map.                                                                                                                                                                                                                                                                                              | `DoScenario = { questID = 86912, mapID = 2505 }`                                                            |
| `LeaveScenario`   | Prompts to exit the scenario if it’s completed but the player is still inside. Usually tied to a quest + map.                                                                                                                                                                                                                                                                       | `LeaveScenario = { questID = 86912, mapID = 2505 }`                                                         |
| `EnterInstance`   | Guides the player to enter an instance (map ID). Typically used when the step must be executed inside the instance.                                                                                                                                                                                                                                                                 | `EnterInstance = { questID = 12345, mapID = 2505 }`                                                         |
| `LeaveInstance`   | Prompts the player to leave an instance (map ID) when objectives are done but the player is still inside.                                                                                                                                                                                                                                                                           | `LeaveInstance = { questID = 12345, mapID = 2505 }`                                                         |
| `SetHS`           | Step to set the Hearthstone.                                                                                                                                                                                                                                                                                                                                                        | `SetHS = 31732`                                                                                             |
| `UseHS`           | Step to use the Hearthstone.                                                                                                                                                                                                                                                                                                                                                        | `UseHS = 31732`                                                                                             |
| `UseDalaHS`       | Dalaran Hearthstone variant.                                                                                                                                                                                                                                                                                                                                                        | `UseDalaHS = 44184`                                                                                         |
| `UseGarrisonHS`   | Garrison Hearthstone variant.                                                                                                                                                                                                                                                                                                                                                       | `UseGarrisonHS = 110560`                                                                                    |
| `UseItem`         | Step requiring the use of a specific quest item.                                                                                                                                                                                                                                                                                                                                    | `UseItem = { questID = 42008, itemID = 173430, itemSpellID = 254294 }`                                      |
| `UseSpell`        | Requires casting a specific spell tied to a quest.                                                                                                                                                                                                                                                                                                                                  | `UseSpell = { questID = 42476, spellID = 193759 }`                                                          |
| `UseFlightPath`   | Step for using a flight master. Validates once the flight is complete.                                                                                                                                                                                                                                                                                                              | `UseFlightPath = 39580`                                                                                     |
| `GetFP`           | Learn a flight path node.                                                                                                                                                                                                                                                                                                                                                           | `GetFP = 2395`                                                                                              |
| `LearnProfession` | Checks if a specific profession spell is learned.                                                                                                                                                                                                                                                                                                                                   | `LearnProfession = 2259`                                                                                    |
| `LootItems`       | Checks for quest items in inventory and displays their name until obtained. Each entry can be tied to a quest ID to keep the step progression consistent.                                                                                                                                                                                                                           | `LootItems = { { questID = 86644, itemID = 244143, quantity = 1 } }`                                        |
| `WarMode`         | Instructs to enable War Mode through the related quest.                                                                                                                                                                                                                                                                                                                             | `WarMode = 60361`                                                                                           |
| `Grind`           | Requires reaching a specific player level before proceeding.                                                                                                                                                                                                                                                                                                                        | `Grind = 60`                                                                                                |
| `LeaveQuest`      | Abandons a single quest from the quest log.                                                                                                                                                                                                                                                                                                                                         | `LeaveQuest = 38254`                                                                                        |
| `LeaveQuests`     | Abandons multiple quests.                                                                                                                                                                                                                                                                                                                                                           | `LeaveQuests = { 38254, 38257 }`                                                                            |
| `VehicleExit`     | Forces exiting a vehicle.                                                                                                                                                                                                                                                                                                                                                           | `VehicleExit = true`                                                                                        |
| `MountVehicle`    | Automatically validates when a mount event is detected.                                                                                                                                                                                                                                                                                                                             | `MountVehicle = true`                                                                                       |
| `NpcDismount`     | Automatically dismount to talk to the npc.                                                                                                                                                                                                                                                                                                                                          | `NpcDismount = 43733`                                                                                       |
| `ResetRoute`      | Displays a popup to reset the current route.                                                                                                                                                                                                                                                                                                                                        | `ResetRoute = true`                                                                                         |
| `Emote`           | Requires performing an emote on a target.                                                                                                                                                                                                                                                                                                                                           | `Emote = "salute"`                                                                                          |
| `ChromiePick`     | Selects a specific timeline from Chromie using its ID.                                                                                                                                                                                                                                                                                                                              | `ChromiePick = 8`                                                                                           |
| `BuyMerchant`     | Prompts the user to buy items from a merchant. Can be used as `main` or `secondary` action.                                                                                                                                                                                                                                                                                         | `BuyMerchant = { { itemID = 193890, quantity = 1, questID = 66680 } }`                                      |
| `RouteCompleted`  | Marks the route as finished and triggers final actions. Must always be the last step of a route.                                                                                                                                                                                                                                                                                    | `RouteCompleted = true`                                                                                     |

## Navigation and Targeting

| Option                 | Description                                                                                                                                                                                                                                                             | Expected Syntax                                                                                                                     |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `Coord`                | World coordinates used by the arrow (`x`, `y` in WoW units).                                                                                                                                                                                                            | `Coord = { x = 4298.4, y = -864.1 }`                                                                                                |
| `Coords`               | Allows multiple coordinates for a single step. Mandatory when using Zones. Each entry must define the related Zone and the x/y position to be used when the player is inside that zone. The arrow dynamically selects the correct coordinate based on the current zone. | `Coords = { { Zone = 84, x = 797.7, y = -8624.9 }, { Zone = 85, x = -4436, y = 1590.3 }, { Zone = 2248, x = -2462.1, y = 2813.4 }}` |
| `Zone`                 | Expected map ID for validation.                                                                                                                                                                                                                                         | `Zone = 627`                                                                                                                        |
| `Zones`                | Allows multiple valid zones for a single step. The step is considered valid if the player is in any of the listed map IDs. Typically used with Coords.                                                                                                                  | `Zones = { 84, 85 }`                                                                                                                |
| `Range`                | Distance in yards from `Coord` to consider the location reached.                                                                                                                                                                                                        | `Range = 45`                                                                                                                        |
| `ZoneStepTrigger`      | Automatically validates the step when entering a defined zone radius.                                                                                                                                                                                                   | `ZoneStepTrigger = { x = 4098.2, y = -712.4, Range = 25 }`                                                                          |
| `Waypoint`             | Quest-related waypoint displayed until completion.                                                                                                                                                                                                                      | `Waypoint = 44543`                                                                                                                  |
| `WaypointDB`           | Alternative waypoint quests that allow skipping if already completed. **Requires the base `Waypoint` field.**                                                                                                                                                           | `WaypointDB = { 44543, 44544 }`                                                                                                     |
| `NonSkippableWaypoint` | Prevents the step from being skipped manually.                                                                                                                                                                                                                          | `NonSkippableWaypoint = true`                                                                                                       |
| `TakePortal`           | Step requiring the player to take a portal. The step validates once the player reaches the target zone after using the portal. Can be used to guide portal transitions between cities/continents.                                                                       | `TakePortal = { questID = 81888, ZoneId = 85 }`                                                                                     |
| `NodeID`               | Flight path node ID used only on steps with `UseFlightPath`.                                                                                                                                                                                                            | `NodeID = 1719`                                                                                                                     |
| `Name`                 | Optional custom name to override the TaxiNode name.                                                                                                                                                                                                                     | `Name = "Krasus' Landing"`                                                                                                          |
| `Boat`                 | Indicates the player should take a boat instead of flying. **Requires a `UseFlightPath` step.**                                                                                                                                                                         | `Boat = true`                                                                                                                       |
| `NoArrow`              | Hides the navigation arrow for this step.                                                                                                                                                                                                                               | `NoArrow = true`                                                                                                                    |
| `NoAutoFlightMap`      | Prevents automatic flight/gossip selection for this step.                                                                                                                                                                                                               | `NoAutoFlightMap = true`                                                                                                            |
| `InstanceQuest`        | Marks step as taking place inside an instance.                                                                                                                                                                                                                          | `InstanceQuest = true`                                                                                                              |
| `IsAdventureMap`       | Allows auto-accepting quests from the Adventure Map.                                                                                                                                                                                                                    | `IsAdventureMap = true`                                                                                                             |
| `ETA`                  | Estimated AFK timer duration (seconds).                                                                                                                                                                                                                                 | `ETA = 75`                                                                                                                          |
| `GossipETA`            | Starts an AFK timer after gossip confirmation.                                                                                                                                                                                                                          | `GossipETA = 45`                                                                                                                    |
| `SpecialETAHide`       | Hides AFK timer even if `ETA` is set.                                                                                                                                                                                                                                   | `SpecialETAHide = true`                                                                                                             |

## Automation and Display

| Option            | Description                                                                                                                                                                                                                                                    | Expected Syntax                                                       |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| `Buffs`           | List of buff spell IDs to recommend.                                                                                                                                                                                                                           | `Buffs = { { spellId = 311103, tooltipMessage = "FRESHLEAF_BUFF" } }` |
| `Button`          | Associates items to use with quest objectives (`"QuestID-Objective"` → `itemID`). Buttons can be used on **Qpart**, **Fillers**, or **DroppableQuest**, and also on other step types (`PickUp`, `Done`) by using only the quest ID without an objective index. | `Button = { ["30778-1"] = 81356 }`                                    |
| `SpellButton`     | Creates a spell button for a quest objective (`"QuestID-Objective"` → `spellID`). Like `Button`, it can be used on **Qpart**, **Fillers**, or **DroppableQuest**, or on other steps (`PickUp`, `Done`) using only the quest ID.                                | `SpellButton = { ["49939-1"] = 294197 }`                              |
| `SpellTrigger`    | Automatically completes the step when the given spell is cast.                                                                                                                                                                                                 | `SpellTrigger = 306719`                                               |
| `ExtraActionB`    | Prompts use of the special action button (bonus bar).                                                                                                                                                                                                          | `ExtraActionB = true`                                                 |
| `TrigText*`       | Text fragments used as completion triggers (`TrigText`, `TrigText2`, …).                                                                                                                                                                                       | `TrigText = "Restore the console"`                                    |
| `Bloodlust`       | Adds a reminder to use Heroism/Bloodlust.                                                                                                                                                                                                                      | `Bloodlust = true`                                                    |
| `InVehicle`       | Indicates vehicle status (`1` = enter, `2` = stay).                                                                                                                                                                                                            | `InVehicle = 1`                                                       |
| `UseGlider`       | Displays available gliders for controlled jumps.                                                                                                                                                                                                               | `UseGlider = true`                                                    |
| `DenyNPC`         | The ID of an NPC whose gossip should be closed automatically (used for anti-error handling).                                                                                                                                                                   | `DenyNPC = 209914`                                                    |
| `RaidIcon`        | The ID of an NPC to mark with a raid icon.                                                                                                                                                                                                                     | `RaidIcon = 241743`                                                   |
| `GossipOptionIDs` | Automates gossip selection by specific option IDs. Use `main` for primary (talk-to-NPC) or `secondary` for follow-up actions like quest turn-ins.                                                                                                              | `GossipOptionIDs = { 51901, 51902 }`                                  |
| `Dontskipvid`     | Optional flag to prevent automatic skipping of cutscenes or videos.                                                                                                                                                                                            | `Dontskipvid = true`                                                  |

## Filters and Conditions

| Option                              | Description                                                                                                         | Expected Syntax                                        |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| `Faction`                           | Restricts step to a specific faction (`"Alliance"` or `"Horde"`).                                                   | `Faction = "Horde"`                                    |
| `Race`                              | Restricts step to listed races.                                                                                     | `Race = { "Orc", "Troll" }`                            |
| `Gender`                            | Restricts to a gender (`1` neutral, `2` male, `3` female).                                                          | `Gender = 3`                                           |
| `Class`                             | Restricts step to specific classes.                                                                                 | `Class = { "HUNTER", "ROGUE" }`                        |
| `HasAchievement`                    | Requires a specific achievement to be completed.                                                                    | `HasAchievement = 12593`                               |
| `DontHaveAchievement`               | Visible only if the achievement is missing.                                                                         | `DontHaveAchievement = 9924`                           |
| `HasAura`                           | Checks if a specific buff is active.                                                                                | `HasAura = 178207`                                     |
| `DontHaveAura`                      | Checks if a specific buff is missing.                                                                               | `DontHaveAura = 32182`                                 |
| `HasSpell`                          | Requires knowing a specific spell (mount, ability, etc.).                                                           | `HasSpell = 34090`                                     |
| `IsOneOfQuestsCompleted`            | Skips the step if any listed quest is completed.                                                                    | `IsOneOfQuestsCompleted = { 31588, 31589 }`            |
| `IsOneOfQuestsUncompleted`          | Displays only if at least one quest is uncompleted.                                                                 | `IsOneOfQuestsUncompleted = { 31588, 31589 }`          |
| `IsOneOfQuestsCompletedOnAccount`   | Checks completion across the account.                                                                               | `IsOneOfQuestsCompletedOnAccount = { 49929, 49930 }`   |
| `IsOneOfQuestsUncompletedOnAccount` | Displays if none of the quests are completed on the account.                                                        | `IsOneOfQuestsUncompletedOnAccount = { 49929, 49930 }` |
| `IsQuestsCompleted`                 | All listed quests must be completed.                                                                                | `IsQuestsCompleted = { 31821, 31822 }`                 |
| `IsQuestsUncompleted`               | At least one listed quest must be incomplete.                                                                       | `IsQuestsUncompleted = { 31821, 31822 }`               |
| `IsQuestsCompletedOnAccount`        | Account-wide version of `IsQuestsCompleted`.                                                                        | `IsQuestsCompletedOnAccount = { 49929, 49930 }`        |
| `IsQuestsUncompletedOnAccount`      | Account-wide version of `IsQuestsUncompleted`.                                                                      | `IsQuestsUncompletedOnAccount = { 49929, 49930 }`      |
| `QuestLineSkip`                     | Prevents group popup if a questline is intentionally skipped.                                                       | `QuestLineSkip = 51226`                                |
| `PickedLoa`                         | Specifies the chosen Loa in Zandalar (`1` = Bwonsamdi, `2` = Rezan).                                                | `PickedLoa = 1`                                        |
| `IsCampaignQuest`                   | Marks the step as part of a campaign.                                                                               | `IsCampaignQuest = true`                               |
| `skipForLvl`                        | Skips the step if your level is **greater than or equal** to the given value. (Often written as “skip if lvl 90+”.) | `skipForLvl = 90`                                      |

## Miscellaneous and Legacy

| Option      | Description                                                                    | Expected Syntax     |
| ----------- | ------------------------------------------------------------------------------ | ------------------- |
| `ExtraLine` | Legacy field for displaying a static localized helper line.                    | `ExtraLine = 13544` |
| `Gossip`    | Legacy field for automatically selecting a gossip option by index.             | `Gossip = 2`        |
| `_index`    | Internal index auto-generated during route packaging (do not modify manually). | `_index = 128`      |

## Example Step

```lua
{
    PickUp = { 39688 },
    PickUpDB = { 39688, 39694, 40255, 40256 },
    Coord = { x = -451.2, y = 4283.7 },
    _index = 113,
},
{
    Qpart = { [30781] = { 1 } },
    Coord = { x = 2205.4, y = 1826 },
    Fillers = { [30778] = { 1 }, [30779] = { 1 }, [30780] = { 1 } },
    Button = { ["30778-1"] = 81356 },
    DroppableQuest = { Qid = 30779, MobId = 60622, Text = "Thunder Hold Laborer" },
    Zone = 388,
    _index = 29,
},
{
    QpartPart = { [40567] = { 1 } },
    TrigText = "2/3",
    Coord = { x = 5112.3, y = 2781.9 },
    Range = 0.75,
    ExtraLineText = "Interact with the second console",
    Zone = 630,
    _index = 44,
},
{
    Done = { 30796, 30797 },
    Coord = { x = 1224, y = 3584.7 },
    Button = { ["30796"] = 81712 },
    ExtraLineText = "USE_THE_TONGUE_BA_SHON_TO_SPAWN_NPC",
    Zone = 379,
    _index = 189,
},
{
    PickUp = { 30799 },
    Coord = { x = 1224, y = 3584.7 },
    Button = { ["30799"] = 81712 },
    ExtraLineText = "USE_THE_TONGUE_BA_SHON_TO_SPAWN_NPC",
    Zone = 379,
    _index = 190,
},
{
    RouteCompleted = true,
}
```
